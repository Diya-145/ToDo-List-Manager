# 📝 To-Do List Manager

A simple command-line To-Do List application built using Python.

## Features

- Add Tasks
- View Tasks
- Delete Tasks
- Mark Tasks as Completed
- JSON Storage

## Technologies Used

- Python
- JSON
- File Handling

## How to Run

```bash
python todo.py
```
