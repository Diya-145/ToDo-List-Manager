# 📝 To-Do List Manager

A command-line task management application built with Python.

## ✨ Features

- ✅ Add Tasks
- ✅ View Tasks
- ✅ Delete Tasks
- ✅ Mark Tasks as Completed
- ✅ Search Tasks
- ✅ Due Dates
- ✅ Statistics Dashboard
- ✅ Export Tasks to CSV
- ✅ JSON Storage

## 🛠 Tech Stack

- Python
- JSON
- CSV
- Colorama

## ▶️ Run

```bash
pip install colorama
python todo.py
```
